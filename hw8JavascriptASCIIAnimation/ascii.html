Instrution to use Custom Input


1. Please stop the animation to enable the text box for custom use.

2. Choose Custom animation type from the drop-down.

3. Type your frames in the text box. Between each frame please enter five equal signs (=====).

4. Finally, please enter start to being the animation.


Note: You may check the format from other animation types. Or you may copy and paste from one of the animation types to try the Custom feature.
